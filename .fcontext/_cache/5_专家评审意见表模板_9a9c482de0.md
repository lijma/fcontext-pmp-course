| 2016年9月19日，XX市财政局组织专家在市财政局对“XX市国库集中支付电子化管理项目”进行验收，与会专家听取了项目建设单位XX市财政局的项目基本情况介绍；项目承建单位北京XXXX信息技术股份有限公司、深圳XX软件有限公司和长春XXXX信息技术股份有限公司的建设情况汇报；经质询与讨论，形成如下验收意见：   1. 提供的验收资料齐全、规范，符合验收要求。 2. 项目按照财政部、人民银行国库集中支付电子化管理规范和有关要求进行设计并建设。试运行以来硬件设备运行稳定，软件系统正常运行，数据交互正常。基本达到了财政部、人民银行所要求的集中支付电子化管理系统建设目标。 3. 各承建单位均按照合同约定基本完成了全部建设内容。   专家组一致同意项目通过验收。  建议：承建单位继续做好维保、培训等工作，并根据业务需求不断完善系统功能。  专家组组长：  验收时间： | | | |
